    search(query_builder) click to toggle source

    Searches the configured search engine with the given query. When adding your own search engine, you may redefine this, as long as it works with the appropriate types. @param [QueryBuilder] query_builder: The query builder you are searching with.

    @return [Array<Result>]: An array of search results (depends on the search engine). In order to map the results, use the ‘result_mapping` block.

    # File lib/dsl.rb, line 57
def search(query_builder)
  map_results(self.class.get(endpoint, query: query(query_builder)))
end
