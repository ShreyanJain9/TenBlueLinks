Overview

A class that your search engine should inherit from. This is the base class for all search engines. By default, this class provides a robust DSL you can use to craft the different pieces of your search engine that must come together to satisfy the requirements of the default #search method. However, you may redefine the #search method to suit your needs, as long as it takes a QueryBuilder as an argument and returns an array of Results.
